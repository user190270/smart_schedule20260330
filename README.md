# Smart Schedule

一个面向移动端场景的本地优先日程系统，集成 AI 解析、知识库问答与分享能力。

## 项目包含什么

- `Vue 3 + TypeScript` 前端
- `FastAPI` 后端
- `PostgreSQL + pgvector` 云端数据链路
- 本地优先日程仓与显式 `Push / Pull`
- 面向日程场景的解析 Agent，用于多轮澄清、草稿维护和确认前辅助生成
- 知识库重建、RAG 问答与分享链路
- `Capacitor Android` 移动端壳，用于真机或模拟器验证

## 当前产品形态

- 日程的创建、编辑、删除围绕本地仓展开。
- 云端同步和知识库重建都是用户可见、可控制的显式动作。
- AI 解析负责理解自然语言、补全缺失字段、维护草稿状态。
- 用户确认后，系统才会把草稿写入本地仓，并按所选策略进入云端和知识库链路。
- 前端统一通过本项目后端访问云端能力与模型能力。
